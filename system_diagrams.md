```
[用户操作] ──→ [Vue前端] ──→ [Flask后端API] ──→ [RecommenderService]
                                      │                    │
                                      ↓                    ↓
                                [SQLite数据库]       [LightGCN模型]
                                      │                    │
                                      └──────→ [返回推荐结果] ←──────┘
                                                    │
                                              [Vue渲染展示]
```
